# RestaurantPlatform
Restaurant Management Application

A console-based Java application to manage menu items, customers, and orders using Java Collections. This project practices CRUD operations, dynamic switch-case menus, and exception handling.

Features

Menu Management: Add, view, update, delete menu items (ArrayList)

Customer Management: Add, view, update, delete customers (HashMap)

Order Management: Create, view, update, cancel orders (HashMap for orders, LinkedList for order items)

Exception Handling: Custom exceptions for invalid input, duplicates, and missing records

Collections Used

ArrayList for menu items

HashMap for customers and orders

HashSet for unique categories

LinkedList for order items
